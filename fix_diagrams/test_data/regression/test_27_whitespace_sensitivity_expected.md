# Whitespace Sensitivity Test

Variant 1 (minimal spaces):
```
┌─────┐──▶┌─────┐
│ Box │   │Box 2│
└─────┘   └─────┘
```

Variant 2 (extra spaces):
```
┌─────┐   ───▶   ┌─────┐
│ Box │           │Box 2│
└─────┘           └─────┘
```

Variant 3 (mixed tabs/spaces):
```
┌─────┐	──▶	┌─────┐
│ Box │		│Box 2│
└─────┘		└─────┘
```
